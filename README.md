# MaanVasam
Mann Vaasam is a responsive Farm-to-Home e-commerce website that connects local farmers directly with customers. The website showcases fresh vegetables, fruits, eggs, dairy products, and other farm products with a modern UI, product categories, promotional banners, and an engaging user experience.


# 🌿 Mann Vaasam – Farm to Home E-Commerce Website

Mann Vaasam is a modern and responsive Farm-to-Home e-commerce website designed to connect local farmers directly with customers. The platform promotes fresh, chemical-free, and naturally grown products while eliminating middlemen.

## ✨ Features

- 🏡 Responsive Home Page
- 🎥 Hero Video Banner
- 🥕 Product Categories Section
- 🛒 Product Listing Cards
- 📖 About Us Section
- 🎯 Mission & Vision Cards
- 🌱 Farm-to-Home Concept
- 📱 Fully Responsive Design
- 🎨 Modern UI/UX Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 📌 Project Status

🚧 Currently under development with additional features like product pages, shopping cart, authentication, and backend integration planned for future updates.

## 📸 Preview

A clean and modern farm-fresh e-commerce interface inspired by real-world grocery delivery platforms.
